# 🛡️ Umbrella Corp - Mission Control System

Sistema de gerenciamento de missões desenvolvido com React e TypeScript.

## 🚀 Justificativa da Arquitetura
O projeto foi estruturado para garantir escalabilidade e manutenção facilitada:
- **Componentização:** A interface foi dividida em componentes menores (`Dashboard` e `CardMissao`), permitindo a reutilização de código.
- **Data & Interfaces:** Os dados foram isolados em arquivos próprios, e o uso de **TypeScript Interfaces** garante que as propriedades das missões sejam seguidas rigidamente em todo o app.
- **Layout:** Utilização de Grid Bootstrap para um design responsivo e assimétrico, separando o status lateral da lista principal.
